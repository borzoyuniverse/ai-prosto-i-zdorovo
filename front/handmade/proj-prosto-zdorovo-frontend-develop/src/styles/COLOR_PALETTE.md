# Цветовая палитра проекта

Этот документ описывает цветовую палитру проекта, основанную на дизайн-системе из Figma.

| `--neutral-0`     | `#FFFFFF` 
| `--neutral-100` | `#EAEAEA` |

| `--neutral-700` | `#6A6A6A` |

## Использование в Tailwind CSS

Цвета доступны через стандартные классы Tailwind с префиксами согласно названиям в палитре:

```html
<!-- Примеры использования цветов -->
<div class="border border-neutral-200">Элемент с рамкой</div>
<div class="text-primary-500">Кнопка основного действия</div>
```
